# Quiz Typescript App

A simple react app with typescript.

### Installing and Executing program

- Clone and install the packages:

  ```
  $ git clone https://github.com/robertovilla2102/quiz-typescript-app
  $ cd quiz-typescript-app
  $ npm install
  ```

  ```
    $ npm start
  ```
